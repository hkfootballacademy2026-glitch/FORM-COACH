# FORM COACH ⚽ — サッカー動作分析アプリ

ブラウザだけで動くサッカー動作分析ツール。キック・トラップ・ドリブル・リフティングの動画を骨格推定AIで解析し、スコアとアドバイスを表示します。

## 使い方

**公開ページで開く**(推奨・カメラ機能あり)
このリポジトリをGitHub Pagesで公開すると、以下のURLで誰でも使えます。
```
https://<あなたのユーザー名>.github.io/<リポジトリ名>/
```

**手元で直接開く**
`index.html` をダブルクリックしてブラウザで開くだけでも動作します(ライブ計測=カメラ機能を除く)。

## 主な機能

- **4モード**: キック / トラップ / ドリブル / リフティング
- **骨格推定**: MediaPipe Pose(ブラウザ内で処理、映像は外部送信されません)
- **ボール検出**: TensorFlow.js + COCO-SSD
- **動画アップロード解析** / **ライブ計測**(カメラで自動検出・自動計測開始)
- **お手本動画との比較**(骨格ゴースト表示・差分テーブル)
- **PRO ZONEレーダーチャート**(競技者水準の目安帯との比較)

## 技術メモ

- 全処理がブラウザ内(クライアントサイド)で完結。サーバー不要、動画は外部送信されません
- ライブ計測にはカメラ権限とHTTPS(またはlocalhost)環境が必要です
- 各種スコアの閾値は一般的なコーチング目安からの仮値です。継続的にチューニングしています
